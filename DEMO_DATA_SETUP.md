## 🛠️ Setup Scripts

### Option 1: Use Backend Seed Script (Recommended)

If you have a seed script in backend:

```bash
cd backend
source venv/bin/activate
python -m src.scripts.seed_demo_data
```

### Option 2: Manual Setup via API

Use the API documentation at `http://localhost:8000/docs` to:

1. Create demo user account
2. Complete profile via PUT `/api/v1/profiles/me`
3. Add skills via POST `/api/v1/profiles/me/skills`
4. Add work experience via POST `/api/v1/profiles/me/experience`
5. Create other user accounts
6. Send connection requests via POST `/api/v1/network/connect/{user_id}`
7. Register for events via POST `/api/v1/events/{event_id}/register`
8. Join communities via POST `/api/v1/communities/{community_id}/join`
9. Create posts via POST `/api/v1/communities/{community_id}/posts`
10. Send messages via POST `/api/v1/conversations/{user_id}/messages`

### Option 3: Use Frontend UI

The most realistic approach:

1. Sign up as demo user
2. Complete 7-step profile wizard
3. Create additional accounts for connections (use temp email addresses)
4. Manually connect, join communities, register events
5. Create posts and comments
6. Send messages between accounts

**Time Required:** 2-3 hours for comprehensive data

## 🚨 Common Setup Issues

**Issue: Profile completion stuck at 90%**
→ Check: Awards and certifications often missing

**Issue: Graph visualization empty**
→ Run: Neo4j sync job or manually create relationships in Neo4j Browser

**Issue: No recommendations showing**
→ Ensure: Profile is complete, skills added, connections exist

**Issue: AI features not working**
→ Check: `OPENAI_API_KEY` in `backend/.env`

**Issue: Events not showing in calendar**
→ Verify: Event dates are in current/future month
